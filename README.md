2- What is the spread operator (...)?
ANS- it is like a deconstruct button for arrays and objects.
     It takes an existing collection and spreads its individual items out into a new location. 

3-What is the difference between map(), filter(), and forEach()?
ANS- forEach() executes a function for each element.
     map() transforms each element into a new array of the same length.
     filter() creates a new array containing only elements that meet a specific condition.
